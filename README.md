# Jekyll Starter

A minimal Jekyll scaffold.

## Requirements

- Ruby (2.7+ recommended)
- Bundler (`gem install bundler`)

## Setup

```bash
bundle install
```

Or:

```bash
make install
```

## Run

```bash
bundle exec jekyll serve --livereload
```

Or:

```bash
make serve
```

Visit http://localhost:4000

## Build

```bash
make
```

This outputs the site to `_site/`.

## Clean

```bash
make clean
```
